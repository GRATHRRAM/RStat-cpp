# RStats C++
### RStats Its A Roblox Game Stats Viewer Written in C++
## Compilation
```bash
# Clone Repo
mkdir build
cd build
cmake .. # On Windows Add -G "MinGW Makefiles" if Youre Using MinGW
make # Add -j(number of Threads) to make building faster
cp ../CopyToBuildFolder/* ./
# Done!
```
